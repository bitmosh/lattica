## To run locally

```bash
cd ~/Projects/lattica
npm install           # install JS deps
npm run tauri:dev     # starts vite on :1421 + tauri dev window
```

Note: `tauri dev` will trigger a `cargo build` which compiles fossic and fossic-tauri
from their path dependencies. First build takes ~2–5 minutes.

## Known bugs

None in new code. Icons directory is absent — `tauri build` will fail until icons are
provided. `tauri dev` works without icons.
