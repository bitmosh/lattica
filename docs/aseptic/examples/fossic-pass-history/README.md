# fossic Pass History — Worked Examples

These files are fossic's actual pass blast-radius reports from its
v0.1.0 through v0.10.x build. They were preserved in Lattica's
docs tree during the v0.0.0 bootstrap (the Aseptic working copy
adopted into Lattica was fossic's, and these files came along).

They are **not** Lattica passes. They are kept here as reference
material — worked examples of what real Aseptic blast-radius
entries look like when the methodology is operating well, what
retroactive filing looks like, how resolution conventions are
followed in practice, and how multi-stage cleanup runs (`pass-10.w`,
`pass-10.x`, etc.) are structured.

Future Lattica passes use the numbering scheme in
`../../blast-radius/` (e.g., `pass-00.md`, `pass-0.0.0z.md`,
`pass-0.1.0.md`). When in doubt about whether a blast-radius file
is Lattica history, check its location: `blast-radius/` is Lattica;
`examples/fossic-pass-history/` is fossic reference.

The canonical fossic Aseptic working copy lives at
`~/Projects/fossic/docs/aseptic/` (if and when it is established
there). The files in this directory are a frozen snapshot from the
bootstrap moment, not a live mirror.
